# Salsa Verde Platform

A mobile and web application for managing plant maintenance operations,
including technician workflows, task tracking, and manager approvals.

## Project Structure

- `apps/mobile` – Expo (React Native) mobile application
- `apps/api` – Node.js Express backend API
- `docs` – Project documentation and diagrams

## Tech Stack

- JavaScript
- Expo (React Native)
- Node.js + Express
- Jira (Scrum project management)

## Status

Project is under active development.
